# 30 Days of Web Development

A learning repository for building HTML/CSS projects over a 7-day challenge.

## Project Overview

This repository contains day-by-day web development practice projects focused on HTML and CSS fundamentals. Each `DAY-x` folder includes small exercises, demos, and mini-projects for learning layout, styling, typography, and responsive page structure.

## Folder Structure

- `HTML/`
  - `DAY-1/` — basic HTML structure and introductory pages
  - `DAY-2/` — additional HTML practice with page content and layout
  - `DAY-3/` — continued HTML exercises
  - `DAY-4/` — HTML with asset organization
  - `DAY-5/` — introduction of CSS with `main.js`, `style.css`, and favicon assets
  - `DAY-6/` — CSS fundamentals: box model, colors, fonts, selectors, specificity, units, and a project folder
  - `DAY-7/` — styling practice with background and project pages

## How to Use

1. Open the `HTML/` folder in a browser or editor.
2. Navigate into a `DAY-x/` folder.
3. Open the `index.html` file for that day.
4. Review or edit the HTML and CSS files to experiment with the layout and design.

## Notes

- This repository is primarily for hands-on web development practice.
- Some days include complete mini-projects inside nested folders such as `DAY-6/project/` and `DAY-7/Project/`.
- Files like `style.css` and `main.js` are used to apply styles and interactive behavior.
